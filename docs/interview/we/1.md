## 测试

1. 对数组求最大值（至少两种实现方法）
2. 解析 URL params 参数

    考虑尽可能全面，注意边界情况。如：

    `let url = 'http://www.domain.com/?user=anonymous&id=123&id=456&city=%E5%8C%97%E4%BA%AC&enabled#detail';`

    执行 `parseParam(url)` 输出

    ```js
    {
      user: 'anonymous',
      id: [ 123, 456 ], // 重复出现的 key 要组装成数组
      city: '北京', // 中文需解码
      enabled: true, // 未指定值得 key 约定为 true
    }
    ```
