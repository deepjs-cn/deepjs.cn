# PWA

## 什么是Progressive Web Apps

PWA(Progressive Web Apps) 不是一个新框架新架构，也不是特指的某一个新概念，而是通过一系列的经过改进后基于web的技术，使 Web App 在安全、性能和体验等方面得到提升，让用户能获得更接近于 Native App 的体验。
